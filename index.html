HoverShock

HoverShock is a fast paced and physics based multiplayer game where the player controls a drone full of weapons and mind-bending powers in an exaggerated and colorful world, full of life, movement and crazy explosions. HoverShock has several game modes designed for a competitive multiplayer experience, taking the drone's handling possibilities to full advantage, like Quiddshock, a quidditch sports mode reimagined for drone clashes.

Raccoon Escape

Raccoon Escape is an endless runner envisioned for mobile platforms. In this game you need to help the Raccoon escape the fiery pits where he is imprisoned! Connect the gears that will pull him up while avoiding various enemies. This is a fast paced endless runner that requires all your attention and skill to succeed!

Digital Aquarium

This was a project developed for Sesimbra's Maritime Museum. It tries to reproduce Sesimbra's maritime wildlife in its natural habitat. It allows its visitors to know more about all the different species that inhabit those waters. This is an interactable rendition of Sesimbra's aquatic life, where each species has its own set of rules to navigate the underwater environment and interact with each other.

Digital Aquarium video

In this project, I was responsible for the environment 3d modeling and composition, as well as some of the aquatic species 3d modeling and rigging. I also did the integration in Unreal Engine 4, all the animal artificial intelligence and rules, as well as all VFX and materials.

Medusa's House VR

This project is a VR experience that allows its user to see what this roman era house looked like at the time. This project was created for the municipality of Alter do Chão and recreates a roman house that now only its foundations remain.

In this project, I did the Unreal Engine 4 integration as well as VR logic and optimizations.

Penela Castle VR

This project is a VR experience that allows its users to see how Penela's castle looked like in the XV century. This project was created for the municipality of Penela and recreates its castle, interior elements, and inhabitants as they were at that time.

Penela's Castle XV Century Presentation Video

This was a solo project with many different responsibilities. Some of those were 3d modeling, texturing, materials, VFX, Unreal Engine 4 integration, VR logic and optimization, character modeling, rigging and artificial intelligence.
